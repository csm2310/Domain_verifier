# Domain_verifier
Overview

The Domain Verifier is a web-based tool built using Flask and HTML that allows users to check the validity and availability of domain names. This system helps users verify whether a domain name is registered, expired, or available for purchase.

Features

Domain name availability check

Whois lookup for registered domains

User-friendly web interface

Fast and accurate domain verification

Technologies Used

Python (Flask framework)

HTML, CSS, JavaScript for frontend

Whois API / Python-whois module for domain lookup

Bootstrap for responsive UI (optional)

Installation

Clone this repository:

git clone https://github.com/your-username/domain-verifier.git

Navigate to the project folder:

cd domain-verifier

Install required dependencies:

pip install -r requirements.txt

Run the Flask application:

python app.py

Open the browser and go to http://127.0.0.1:5000/

Usage

Enter the domain name you want to verify.

Click on the Check button.

View the results to see if the domain is available or taken.
